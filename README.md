# 📊 Data Analytics Project 4 – Data Visualization with Power BI

## 📖 Overview

This project focuses on transforming an E-Commerce dataset into an interactive Power BI dashboard that communicates meaningful business insights through effective data visualization and storytelling.

The dashboard follows data visualization best practices, using appropriate chart selection, KPI cards, interactive filters, and business-focused insights to support informed decision-making.

---

## 🎯 Objectives

* Design an interactive Power BI dashboard
* Apply data storytelling principles to communicate insights
* Analyze sales performance, customer behavior, marketing channels, and order fulfillment
* Present key business metrics through clear and professional visualizations

---

## 📌 Dashboard Highlights

* Executive Overview with KPI Cards
* Monthly Revenue Trend Analysis
* Product Performance Analysis
* Customer Acquisition & Referral Analysis
* Order Fulfillment & Status Analysis
* Customer Purchase Behavior Analysis
* Interactive Slicers for dynamic filtering

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel

---

## 🗂️ Dataset

The project uses a cleaned E-Commerce Orders dataset containing sales transactions, product details, customer information, payment methods, referral sources, and order status records.
